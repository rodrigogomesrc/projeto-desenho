void interpretar ( comando *  entrada,
comando *  ultima_entrada,
imagem *  img 
)

Função de interpretação dos comandos.

Essa função recebe as informações de uma linha de comando representando as primitivas e chama a função da primitiva correspondente com os parâmetros necessários

Parameters
entradarecebe um ponteiro de uma struct comando com as informações da instrução a ser executada.
ultima_entradarecebe um ponteiro de uma struct comando com a ultima primitiva salva *.
imgrecebe um ponteiro dde uma struct imagem com a imagem a ser modificada pelas primitivas

(*) ultima_entrada somente armazena uma primitiva quando corresponde a line ou polygon, para uso das funções repeat_line e repeat_polygon
